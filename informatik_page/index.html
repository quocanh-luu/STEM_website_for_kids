Was ist Programmierung?

Programmieren bedeutet, einem Computer genau zu sagen, was er tun soll. Du schreibst Anweisungen (Code), und der Computer führt sie aus.

Ein Beispiel: Um einen Computer anzuweisen, ein Bild anzuzeigen, könnte der Code so aussehen:

ZeigeBild("sun.jpg");

Was sind Schleifen?

Eine Schleife wiederholt eine Anweisung immer wieder. Zum Beispiel: Wenn du möchtest, dass der Computer 10 Mal "Hallo" sagt, kannst du eine Schleife verwenden:

für (i = 1; i <= 10; i++) {
console.log("Hallo");
}

In diesem Beispiel sagt der Computer 10 Mal "Hallo".

Was sind Bedingungen?

Mit Bedingungen kannst du dem Computer sagen, dass er nur etwas tun soll, wenn eine bestimmte Bedingung erfüllt ist. Beispiel:

wenn (alter >= 18) {
console.log("Du darfst Auto fahren!");
} sonst {
console.log("Du bist noch zu jung!");
}

Hier prüft der Computer, ob du alt genug bist, um Auto zu fahren.

Was sind Variablen?

Variablen sind wie kleine Kisten, in denen du Informationen speichern kannst. Beispiel:

var alter = 10;
var name = "Max";

In diesem Beispiel speicherst du die Zahl 10 in der Variablen "alter" und den Namen "Max" in der Variablen "name".
